# Flight Finder

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your Amadeus API credentials in main directory:

- AMADEUS_API_KEY = your_key

- AMADEUS_API_SECRET = your_secret

4. Create a `.env`file with your Gemini API key in travel-itinerary directory:

- GEMINI_API_KEY= your_key

5. Run: 

- Type `npm install`

- After installation done, type `npm run dev`

- After we get localhost, click the link to go into it

- Come back to vs code, make new terminal then type `cd travel-itinerary`

- Then type `node server.js` to run the Gemini API

- Then we can search on the website
